2. Propósito de esta web
Esta web ha sido creada con el propósito de ayudar a recuperar bicicletas robadas. Es equiparable a un tablón de anuncios o censo de bicicletas robadas donde los propietarios, de forma voluntaria, pueden colgar información acerca de una bici de su propiedad que haya sido sustraída. Estas bicis se añaden a una base de datos en la que se puede buscar de forma cómoda y rápida, para que quien crea haber localizado una bicicleta robada pueda encontrarla rápidamente en el censo.
El propietario puede en todo momento marcar una bicicleta como recuperada, o eliminarla completamente del sistema.

3. Condiciones de uso
Esta web ofrece un servicio similar a un tablón de anuncios, donde usuarios de Argentina publican los datos de sus bicicletas robadas, y otros usuarios pueden buscar dichas bicicletas en la base de datos con el único fin de informar al propietario y tratar de recuperar los vehículos sustraídos.
El fin último de este website es el de tratar de facilitar la recuperación de bicicletas robadas, así como disuadir de su robo.
El usuario que hace uso de este servicio asume las condiciones de uso del mismo. El usuario es el único responsable de los datos introducidos en el sistema, especialmente de los datos referidos a la descripción de la bicicleta y sus accesorios, número de serie y datos de la denuncia oficial del robo.
Al usuario se le solicita una dirección de correo electrónico para registrar una nueva bicicleta robada en el censo. Esta dirección de correo nunca se hará pública, ni se comunicará a terceros ni será usada para otro fin que el del funcionamiento normal del servicio. Esto significa que el usuario podrá recibir comunicaciones desde esta web exclusivamente para:
Enviarle los comentarios de otro usuario que pueda haber localizado su bicicleta
Enviarle el código olvidado de una de sus bicicletas en el censo
Enviarle posibles avisos acerca del funcionamiento de la web o del servicio
Cuando un usuario desea contactar con otro a fin de informarle de una posible pista o localización sobre su bicicleta, se le solicitará una dirección de correo electrónico. Esta dirección no se almacena en el sistema, pero sí se enviará al propietario de la bicicleta sustraída, a fin de que entre ambos usuarios se pueda establecer una comunicación vía email y externa a este website a fin de tratar de localizar y recuperar la bicicleta robada.
El usuario podrá en cualquier momento eliminar del sistema las bicicletas introducidas en el censo.
